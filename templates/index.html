Welcome to Music Player

This document describes the features and API calls available in the Music Player application.

Our Music Player application provides a seamless experience for managing and enjoying your music library. Below is a comprehensive guide to the available functionalities.

1. Show Library

The Show Library API call retrieves all music files available in your pre-set music folder. The library includes all recognized audio formats, organized in a user-friendly manner.

GET /api/library

Response:

  • 200 OK - A JSON array of music files, including metadata like title, artist, album, and duration.
  • 404 Not Found - If the music folder is empty or the path is incorrect.

2. Choose Playlist

The Choose Playlist API call allows users to select an existing playlist to manage or play.

POST /api/playlist/select

Parameters:

  • playlist_name (string) - The name of the playlist you want to select.

Response:

  • 200 OK - The selected playlist details.
  • 404 Not Found - If the playlist does not exist.

3. Play Music

The Play API call plays a specific song from your library or playlist.

POST /api/play

Parameters:

  • song_id (string) - The unique identifier of the song you want to play.

Response:

  • 200 OK - The music starts playing.
  • 404 Not Found - If the song does not exist in the library or playlist.

4. Stop Music

The Stop API call stops the currently playing music.

POST /api/stop

Response:

  • 200 OK - The music stops playing.
  • 400 Bad Request - If no music is currently playing.

5. Add to Playlist

The Add to Playlist API call adds a specific song to the chosen playlist.

POST /api/playlist/add

Parameters:

  • playlist_name (string) - The name of the playlist to add the song to.
  • song_id (string) - The unique identifier of the song to be added.

Response:

  • 200 OK - The song is added to the playlist.
  • 404 Not Found - If the song or playlist does not exist.

6. Remove from Playlist

The Remove from Playlist API call removes a specific song from the chosen playlist.

DELETE /api/playlist/remove

Parameters:

  • playlist_name (string) - The name of the playlist to remove the song from.
  • song_id (string) - The unique identifier of the song to be removed.

Response:

  • 200 OK - The song is removed from the playlist.
  • 404 Not Found - If the song or playlist does not exist.

Happy Listening!
